# WP Synapse AI Lite – The Ultimate WordPress IDE & File Manager

**WP Synapse AI Lite** transforms your WordPress dashboard into a professional-grade development environment. Built for developers who demand speed, reliability, and precision, it brings the power of a desktop IDE directly to your browser.

## 🚀 Key Value Propositions

### 1. Professional VS Code Experience
Powered by the **Monaco Editor**, the same engine that drives VS Code. Enjoy full syntax highlighting for PHP, JS, CSS, and HTML, along with multi-cursor support, smart indentation, and customizable font sizes.

### 2. High-Performance Global Search
Stop hunting for code. Our advanced search engine uses server-native `grep` to scan your entire site instantly.
*   **Search Types**: Switch between Filename Search, Code Search, or both.
*   **Contextual Snippets**: See matching code lines before opening the file.
*   **Deep Linking**: Click a search result to jump directly to that exact line in the editor.

### 3. Safety First: Visual Diff Mode
Never break a live site again. Our built-in **Diff Mode** allows you to compare your current changes with the original file content side-by-side. Review your logic and verify your edits before hitting "Save."

### 4. Enterprise File Management
A familiar, high-speed file tree interface that supports all professional workflows:
*   **Advanced Operations**: Move, Duplicate, Rename, and Delete with ease.
*   **Archive Power**: Create ZIP backups of folders or extract archives directly on the server.
*   **Smart Uploads**: Drag and drop files directly into any directory.

### 5. Developer-First UI/UX
*   **Theme Persistence**: Sleek Dark Mode for late-night sessions and high-contrast Light Mode for daylight coding.
*   **Persistent Context**: Your open tabs and search results stay exactly where you left them, even after a page refresh.
*   **Instant Feedback**: A standardized Toast notification system provides professional success/error feedback for every action.

## 🛠 Technical Specifications
*   **Frontend**: Built with **React 18** and **Framer Motion** for a smooth, high-performance SPA experience.
*   **Icons**: Integrated with **Lucide React** for a modern SaaS aesthetic.
*   **Backend**: Optimized PHP API with secure REST endpoints and native shell integration for heavy-lifting tasks like `grep`.
*   **Performance**: Zero-lag file tree navigation and lightning-fast file saving.

## 📦 Why Choose Synapse Lite?
WP Synapse AI Lite is not just a plugin; it's a productivity booster. It eliminates the need for slow FTP clients and fragmented file managers, giving you a unified, high-speed workspace to build better WordPress sites, faster.
