A powerful AI-assisted code editor for WordPress (Lite Version). Synapse Lite provides a professional IDE experience directly inside your WordPress admin dashboard, allowing you to manage files and folders with ease.

== Description ==

Synapse Lite – AI Code Editor is a modern, high-performance file manager and code editor for WordPress developers. It brings a VS Code-like experience to your WordPress site, enabling you to edit theme and plugin files securely and efficiently.

Key Features:
* **Advanced Code Search**: High-performance "grep" search across all files. Find any hook, function, or variable instantly and jump directly to the exact line in the editor.
* **Smart Monaco Editor**: The same professional engine behind VS Code. Features syntax highlighting, multi-cursor editing, and persistent zoom settings.
* **Visual Diff Mode**: Compare your current changes with the original file content side-by-side before saving. Never lose track of your edits again.
* **Professional File Manager**: Create, move, duplicate, and rename files or folders with a high-speed, intuitive interface.
* **ZIP & Archive Management**: Compress folders to ZIP or extract archives directly on your server.
* **Permissions Manager**: One-click "Safe Fix" for WordPress file system permission issues.
* **Theme Persistence**: Sleek Dark and Light modes that automatically save your preference.
* **Deep Linking**: Search results and file navigation are linked together for a seamless VS Code-style experience.

== Installation ==

1. Upload the `wp-synapse-ai-lite` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Synapse Lite' in your admin sidebar to start coding.

== Frequently Asked Questions ==

= Is this plugin free? =
Yes, the Lite version is a fully-featured, professional IDE and file manager for WordPress.

= Does it require an API key? =
The core file management and search features work locally. Some AI assistance features are powered by external APIs to ensure high-quality output.
