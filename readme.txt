A powerful AI-assisted code editor for WordPress (Lite Version). Features advanced global code search (grep), visual side-by-side Diff mode, Monaco Editor (VS Code engine), and enterprise file management.

== Description ==

Synapse Lite is the ultimate professional IDE for WordPress. Built on the Monaco Editor (the engine behind VS Code), it brings a desktop-class coding experience directly into your WordPress dashboard.

**Key Features:**
* **Monaco Editor Engine**: Experience the same performance and features as VS Code.
* **Global Search (Grep)**: Search for code across your entire WordPress installation instantly.
* **Side-by-Side Diff**: Visually compare changes before saving.
* **Enterprise File Manager**: Manage themes and plugins with professional tools.
* **AI Ready**: Designed to work with Synapse AI for intelligent code suggestions.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-synapse-ai-lite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Access the editor from the 'Synapse Lite' menu item in the dashboard.
