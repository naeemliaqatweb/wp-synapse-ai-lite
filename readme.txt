A powerful AI-assisted code editor for WordPress (Lite Version). Synapse Lite provides a professional IDE experience directly inside your WordPress admin dashboard, allowing you to manage files and folders with ease.

== Description ==

Synapse Lite – AI Code Editor is a modern, high-performance file manager and code editor for WordPress developers. It brings a VS Code-like experience to your WordPress site, enabling you to edit theme and plugin files securely and efficiently.

Key Features:
* **Advanced Code Editor**: Powered by Monaco Editor (the same engine behind VS Code).
* **Intuitive File Explorer**: Manage your WordPress file system with a familiar sidebar interface.
* **Dark/Light Mode**: Switch between themes for comfortable coding in any environment.
* **Lite AI Integration**: Assistance for basic coding tasks and planning.
* **Pro Ready**: Easily upgrade to the Pro version for full autonomous AI agent capabilities.

== Installation ==

1. Upload the `wp-synapse-ai-lite` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Synapse Lite' in your admin sidebar to start coding.

== Frequently Asked Questions ==

= Is this plugin free? =
Yes, the Lite version is fully functional as a code editor and file manager.

= Does it require an API key? =
The Lite version uses local processing for file management. Some AI features may require a connection to the Synapse platform.
